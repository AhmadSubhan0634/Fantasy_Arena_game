# Fantasy Arena

Fantasy Arena is a 2-player turn-based battle game developed in **Python** using **Pygame**. Players choose their favorite anime-inspired fighters and battle inside a customizable arena using unique attacks, healing abilities, and powerful special moves.

---

## Features

- 🧙 Three playable characters:
  - Gojo
  - Sukuna
  - Goku

- ⚔️ Turn-based combat system
- ❤️ Health bars with real-time updates
- 💥 Unique attacks and special abilities for each character
- 🏟️ Custom arena name and environment selection
- 🌋 Multiple environments:
  - Fire
  - Ice
  - Jungle
- 📊 Character comparison before battle
- 🖼️ Dynamic character images based on health status
- 🎨 Fullscreen graphical user interface using Pygame
- 🖱️ Mouse and keyboard controls

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/AhmadSubhan0634/Fantasy_Arena_game.git
```

### 2. Navigate into the project

```bash
cd Fantasy_Arena_game
```

### 3. Install Pygame

```bash
pip install pygame
```

### 4. Run the game

```bash
python fantasy_arena.py
```

## 🎮 How to Play

1. Launch the game.
2. Player 1 selects a fighter.
3. Player 2 selects a fighter.
4. Enter an arena name.
5. Select an arena environment.
6. Optionally compare both characters.
7. Battle begins!
8. Players alternate turns using:
   - Attack
   - Heal
   - Special Ability
9. Reduce your opponent's health to zero to win.
